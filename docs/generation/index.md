# Generation

The three creative domains of the site. Each maps to a top-of-nav page.

- **[Text](text/index.md)** (`/chain`) — multi-step chain jobs that run prompts through an LLM, with optional voice and context steps.
- **[Audio](audio/index.md)** (`/voice`) — voice design, sample cloning, multi-segment synthesis, and the prompts that drive auto-segmentation.
- **[Visual](visual/index.md)** (`/image`) — text-to-image generation through ComfyUI workflows.

All three submit jobs to the same job queue and surface them in **[Jobs](../management/jobs.md)**.
