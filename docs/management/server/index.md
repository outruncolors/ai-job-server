# Server

The Server page (`/server`) is split into three tabs:

- **[Web](web.md)** — resource usage, job counts, and the restart button for *this* FastAPI server
- **[LLM](llm.md)** — OpenAI-compatible endpoint presets that back chain LLM steps and voice auto-segmentation
- **[ComfyUI](comfyui.md)** — start / stop / restart and configure the ComfyUI subprocess

The three tabs touch unrelated subsystems but are grouped here because they're all about the server *running* the work, not the work itself.
